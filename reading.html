Introduction

In this module you will begin learning the basics of JavaScript by reading the textbook and doing coding assignments.

 

Learning Objectives

When you have completed this module you will be able to:

  • Understand the JavaScript concepts:
    • The console.log method
    • Comments
    • Literals
    • Reserved words
    • Use of semicolons
    • Numbers & Arithmetic
    • Strings, String Literals
    • Template Literals
    • Boolean values
    • Empty values - null & undefined
    • Type conversion
    • Variables
